## 1. 营销系统

**核心问题**：如何系统化开展营销？

### 1.1 营销核心定义

**营销**：以与组织目标相一致的方式识别并满足人类与社会的需求。

**营销管理**：选择目标市场并通过创造、传递和沟通卓越的顾客价值，来获取、维持和发展顾客的艺术和科学。

### 1.2 5C分析框架

**识别目标市场的五个因素**：

1. **Customer（顾客）**：目标顾客是谁？需求是什么？
2. **Company（公司）**：我们的能力和资源是什么？
3. **Collaborator（合作者）**：谁能帮助我们？（供应商、渠道、伙伴）
4. **Competitor（竞争者）**：竞争对手是谁？他们的策略？
5. **Context（环境）**：宏观环境因素（政治、经济、社会、技术）

### 1.3 营销战略制定

#### 战略与战术

- **战略**：选择目标市场，决定价值主张
- **战术（营销组合4P）**：实现战略的具体手段

#### 4P营销组合

1. **Product（产品）**
   - 产品种类、质量、设计、特性、品牌、包装、规格、服务、退货

2. **Price（价格）**
   - 定价、折扣、付款条件、融资选择

3. **Place（渠道）**
   - 渠道、覆盖、品类、位置、库存、运输

4. **Promotion（推广）**
   - 广告、促销、公关、人员销售、直销、在线营销

### 1.4 全方位营销

**四大支柱**：

1. **关系营销**：与关键利益相关者建立长期关系
   - 客户、员工、合作伙伴、投资者

2. **整合营销**：协调所有营销活动传递一致价值
   - 各渠道信息一致
   - 活动相互增强

3. **内部营销**：雇佣、培训、激励员工服务客户
   - 所有部门围绕客户目标协作

4. **绩效营销**：了解营销活动的财务和非财务回报
   - 多维度评估：销售、品牌、客户终身价值

### 1.5 新营销现实

**四大市场力量**：
1. 技术：电商、移动、AI
2. 全球化：世界成为公平竞技场
3. 物理环境：气候变化、健康状况
4. 社会责任：可持续、公平、道德

**三大市场结果**：
1. 消费者新能力：随时获取信息、分享观点
2. 公司新能力：精准营销、数据分析
3. 新竞争环境：去管制、私有化、去中间商

### 1.6 营销管理核心案例

**案例：星巴克的营销系统**
- **5C分析**：Customer（追求品质体验的咖啡消费者）、Company（优质咖啡豆+第三空间体验）、Collaborator（全球咖啡农场）、Competitor（本地咖啡馆、连锁品牌）、Context（咖啡文化复兴、城市化）
- **价值主张**：不只是咖啡，而是"第三空间"——家和办公室之外的社交场所
- **4P策略**：Product（手工调制咖啡+空间体验）、Price（溢价定价，强调价值而非价格）、Place（高客流地段+移动端点单）、Promotion（社区营造+个性化杯面+数字化会员）
- **结果**：从西雅图一家小店成长为全球品牌，创造了一个市场并独自占有它

**案例：苹果的整合营销**
- **关系营销**：建立Apple生态锁定（设备间无缝衔接，换品牌成本极高）
- **整合营销**：从产品发布会的戏剧性→零售店的体验设计→广告的极简美学，传递一致的价值
- **内部营销**：Apple Store员工不是"销售员"而是"专家"和"创意顾问"
- **绩效营销**：不只看销售额，更看客户终身价值（LTV）、生态内跨设备购买率

**案例：营销失败的通用汽车**
- **问题**：品牌过多（雪佛兰、别克、凯迪拉克等）互相内耗，定位重叠
- **教训**：没有做5C分析中的Competitor分析——竞争对手不是外部品牌，而是自己内部的品牌
- **启示**：营销管理不是"做更多"，而是"选对战场，做深做透"

### 1.7 营销管理实战框架

**营销诊断四问**：
1. **谁是我们真正的客户？**（不是"所有人"，而是最核心的那群人）
2. **他们为什么选我们而不选竞品？**（差异化到底在哪？）
3. **我们为客户创造了什么独特价值？**（不只是产品功能，而是整体价值主张）
4. **客户从哪里知道我们、如何购买、为什么复购？**（获客→激活→留存→推荐的完整链路）

**营销战略与战术的关系**：
- 战略决定"在哪个战场打"（选择目标市场和价值主张）
- 战术决定"怎么打"（4P营销组合）
- **常见错误**：只有战术没有战略——疯狂做4P但没想清楚5C

## 2. 品牌资产

**核心问题**：如何建立和管理强势品牌？

### 2.1 基于顾客的品牌资产（CBBE模型）

**定义**：品牌资产源于顾客对品牌的认知和感受，是顾客记忆中的品牌知识结构。

**品牌资产的三个关键成分**：

1. **差异性**：顾客对品牌反应不同
2. **品牌知识**：差异由品牌知识导致
3. **顾客反应**：体现在知觉、偏好和行为上

### 2.2 创建强势品牌的四个步骤

#### 步骤一：建立适当的品牌形象

**品牌认知**：
- 品牌识别：认出品牌
- 品牌回忆：想起品牌
- 顶部心智：首先想到的品牌

#### 步骤二：创造适当的品牌含义

**品牌形象的两个维度**：

1. **品牌性能**：
   - 产品可靠性、耐用性、服务效率
   - 与产品相关的联想

2. **品牌形象**：
   - 品牌个性、用户形象、使用场景
   - 与非产品相关的联想

#### 步骤三：引起顾客对品牌的正面反应

**品牌判断**：
- 质量：产品好坏评价
- 信誉：品牌信任度
- 考虑：是否购买意向
- 优越性：相对其他品牌的优势

**品牌感受**：
- 温暖：品牌的社会情感
- 有趣：品牌的娱乐价值
- 兴奋：品牌的刺激感
- 安全：品牌的可靠感
- 社会认同：品牌带来的归属感
- 自尊：品牌带来的自我价值

#### 步骤四：创建与顾客的品牌联系

**品牌共鸣**：
- 行为忠诚：重复购买
- 态度依附：心理依赖
- 社区归属：品牌社区参与
- 主动参与：主动传播和互动

### 2.3 品牌要素选择

**品牌要素**：品牌名称、URL、标识、图标、形象代表、广告语、广告曲、包装

**选择标准**：
1. **可记忆性**：容易回忆和识别
2. **有意义性**：传达品牌信息
3. **可爱性**：内在吸引力
4. **可转移性**：跨品类、跨地域适用
5. **可适应性**：随时间灵活调整
6. **可保护性**：法律保护

### 2.4 品牌定位

**四个要素**：

1. **目标消费者**：品牌面向谁？
2. **参照系**：与谁比较？
3. **差异点**：为什么我们更好？
4. **相信理由**：凭什么相信？

**案例：惠普品牌变革**
- **原定位**：企业级市场，技术导向
- **新定位**：消费电子市场，个人化导向
- **结果**：超越戴尔、IBM，个人消费市场大幅增长

### 2.5 品牌联想三要素

#### 1. 联想强度
- 信息加工的深度和质量
- 相关性越高，联想越强

#### 2. 联想偏好性
- 适应消费者需求
- 成功传递给消费者

#### 3. 联想独特性
- 与众不同
- 提供比较优势

## 品牌与营销的联动路径

1. **品牌→传播→营销**：CBBE建立品牌认知 → STEPPS设计传播 → 4P落地营销组合
2. **营销→品牌**：5C分析选市场 → 4P执行 → 品牌资产沉淀（CBBE的共鸣层）
3. **常见断裂**：品牌定位做了但传播机制没设计 → 品牌停在"认知层"到不了"共鸣层"
