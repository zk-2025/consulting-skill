# 场景：我要系统做营销

> **典型用户**：营销负责人、市场总监、创业者
> **典型开场**："怎么做营销" / "营销没有体系" / "东一榔头西一棒"

---

## 追问引导

1. "你现在做的营销活动，解决了什么业务问题？"
2. "你的目标客户是谁？不是'所有人'，最核心的那群人"
3. "从用户知道你到付费，中间经历了哪些步骤？哪一步流失最多？"
4. "你的4P（产品/价格/渠道/推广）是一致的吗？还是各做各的？"

---

## 诊断路径

```
用户说："我要系统做营销"
  → engine/problem-dissolving.md 追问
  → 常见真问题分支：

A. "没有框架" → modules/brand-and-marketing.md（5C分析+4P组合+全方位营销四大支柱）
B. "有战术无战略" → modules/strategy.md（先锁定问题和战略方向）→ modules/brand-and-marketing.md
C. "传播没设计" → modules/spread-and-growth.md（STEPPS设计传播机制）
D. "转化没说服" → modules/influence.md（在关键触点应用影响力原则）
```

---

## 常见误区

1. **只有4P没有5C**：疯狂做营销但没想清楚客户/竞争/环境 → 先做5C分析
2. **品牌互相内耗**：多个品牌定位重叠 → 每个品牌必须有清晰的差异化
3. **传播和品牌脱节**：传播内容与品牌定位不一致 → 整合营销：各渠道传递一致价值
4. **只看短期收入**：关键结果全是收入指标 → 需要平衡用户指标/收入指标/满意度指标

---

## 对标案例

- **星巴克**：5C分析精准（追求品质的咖啡消费者+第三空间体验+全球咖啡农场）→ 4P高度一致（手工调制+溢价定价+高客流选址+社区营造）→ 从一家小店到全球品牌
- **通用汽车（反面）**：品牌过多（雪佛兰、别克、凯迪拉克）互相内耗 → 教训：没做5C中的Competitor分析——竞争对手其实是自己内部的品牌
- **苹果整合营销**：产品发布会的戏剧性+零售店的体验设计+广告的极简美学+全渠道传递一致价值
